Building this portfolio with Claude Code [SHIPPED]
What I tried

Built a static portfolio site from scratch using Claude Code — no framework, just HTML, CSS, and a touch of vanilla JS. The brief was: minimal, 90s aesthetic, Obsidian-editable resume. Treated it as a real AI ops experiment: Claude as collaborator, not autocomplete.

What I learned

The most useful thing Claude did wasn't write code — it was holding context across decisions (aesthetic direction, content strategy, file structure) so I didn't have to. The failure mode is when you let it make too many micro-choices without a clear constraint. Strong opinions up front = much better output.
